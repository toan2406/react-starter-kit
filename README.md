React Starter Kit is a boilerplate for web development, built on top of Express, React and Redux.

## Features
- [x] Optimized production build
- [x] Code splitting
- [x] Server side rendering
- [x] CSS modules
- [x] Flow type checker
- [x] I18n
- [x] Offline support
